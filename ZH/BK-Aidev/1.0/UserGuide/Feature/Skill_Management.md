# Skill管理

【智能体开发】-》【Skill】tab下可以查看 “全部/当前空间/我的Skill”，所有Skill均可以下载至本地，以供IDE中使用。

![2026-03-11-16-21-16.png](../assets/2026-03-11-16-21-16.png)

支持快捷安装至本地IDE，或以对话方式安装至OpenClaw。
![Skill_Management_20260424102708.png](../assets/Skill_Management_20260424102708.png)

创建Skill只需将技能文件打包为zip上传即可，注意根目录必须包含skill.md文件。

Skill支持版本管理，可以回滚至任意已发布版本。

![Skill_Management_20260423185205.png](../assets/Skill_Management_20260423185205.png)
