# 构建知识库

平台产品【AI开发助手】-》【智能体开发】tab下可以查看 “全部知识/当前空间/我的知识”，可以进行知识的管理、预览。

## 添加知识

【添加知识】入口

![2026-03-13-15-16-46.png](../assets/2026-03-13-15-16-46.png)

知识支持三种创建方式：文件上传、手工录入、网页知识

![2026-03-13-15-21-03.png](../assets/2026-03-13-15-21-03.png)

- 文件上传：选择支持格式的文档文件上传，支持批量上传多个文件
  
  - 非结构化文件
    每个文件均可分别配置知识处理和切片方式

    ![2026-03-13-15-21-15.png](../assets/2026-03-13-15-21-15.png)
  
  - 结构化文件
    目前支持csv、xlsx两种结构化文件格式，每个文件都需要设置向量索引和标量索引字段
    向量索引用于语义检索，标量索引用于数值、固定文案的检索

    ![2026-03-13-15-21-39.png](../assets/2026-03-13-15-21-39.png)
    
    关于知识库索引的使用示例，请查看 [AIDEV知识库索引设置最佳实践](../UserCase/Knowledge_Base_Index_Settings.md)

- 手工录入：直接输入知识的内容文本

![2026-03-13-15-33-33.png](../assets/2026-03-13-15-33-33.png)

- 网页知识：填写知识地址，会拉取知识以供预览，可以指定刷新频率

![2026-03-13-15-33-44.png](../assets/2026-03-13-15-33-44.png)
  
  以iwiki为例，初次绑定时会提示没有权限，点击预览后填写授权页面，待审批通过即可

  ![2026-03-13-15-34-12.png](../assets/2026-03-13-15-34-12.png)

  ![2026-03-13-15-34-27.png](../assets/2026-03-13-15-34-27.png)
  
  按需选择授权用途，如果数据来源超过5个空间，请选 iWiki OpenAI 方式

  ![2026-03-13-15-34-39.png](../assets/2026-03-13-15-34-39.png)
