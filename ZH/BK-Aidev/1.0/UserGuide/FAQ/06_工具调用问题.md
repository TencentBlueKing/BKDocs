# 工具调用问题

## Q：MCP 工具调用失败

A：MCP 工具调用失败应该把报错信息显示出来，并返回给大模型，让大模型理解错误原因并在下次调用时修改正确。

**解决方案：** 已加上错误信息显示功能

## Q：qwen3 调用 MCP 工具出错

A：目前已升级为 qwen3-235B

## Q：gptoss-120b 调用工具生成参数不稳定

A：gptoss-120b 调用工具生成参数不稳定，改用原生 function calling 调用工具会有 responded 和调用结果，导致会有两个回答。

**解决方案：** 目前已通过 stream 流的适配，支持 gptoss-120b 原生 function calling
