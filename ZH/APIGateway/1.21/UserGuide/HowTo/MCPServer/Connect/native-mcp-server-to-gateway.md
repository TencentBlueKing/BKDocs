# 原生的 MCP Server 接入网关

## 说明

原生的 MCP Server 接入网关只是作为普通的 api 接入，暂时不会在网关的 mcp 市场展示

普通的 MCP Server 服务（sse/streamableHTTP 协议等）是可以接入网关的，复用网关的认证/限流等功能。
