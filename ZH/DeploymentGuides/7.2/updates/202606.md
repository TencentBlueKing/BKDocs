## 标准运维

### bk_sops-3.34.11.2186

这是 **安全更新**，修复了安全问题，请尽快安排更新。
<!-- 版本日志见 GitHub_URL 。-->

#### 在中控机更新
在 **中控机** 下载：
``` bash
bkdl-7.2-stable.sh -ur latest bk_sops=3.34.11.2186
```
在 **中控机** 更新：
``` bash
cd $INSTALL_DIR/blueking/  # 进入工作目录
scripts/setup_bkce7.sh -i sops -f
```
#### 在浏览器里下载并更新
下载适用于蓝鲸 7.x 的安装包：
* [bk_sops-V3.34.11.2186.tar.gz](https://bkopen-1252002024.file.myqcloud.com/saas-paas3/bk_sops/bk_sops-V3.34.11.2186.tar.gz)

参考 《[更新安装包](../manual-install-saas.md#更新安装包)》文档上传安装包，无需调整环境变量，参考《[部署标准运维（bk_sops）](../manual-install-saas.md#部署标准运维（bk_sops）)》文档重新部署。

部署成功后，即可在桌面访问了。
