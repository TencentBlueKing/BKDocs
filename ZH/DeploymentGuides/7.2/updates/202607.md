## 容器管理平台

### bcs-services-stack-1.29.6

这是 **安全更新**，修复了安全问题，请尽快安排更新。
<!-- 版本日志见 GitHub_URL 。-->

登录到 **中控机**，先更新 helm 仓库缓存：
``` bash
helm repo update
```
检查仓库里的版本：
``` bash
helm search repo bcs-services-stack --version 1.29.6
```
预期输出如下所示：
>``` plain
>NAME                         CHART VERSION  APP VERSION  DESCRIPTION
>blueking/bcs-services-stack  1.29.6         略           略
>```

接下来开始升级了。

先进入工作目录：
``` bash
cd $INSTALL_DIR/blueking/  # 进入工作目录
```

修改 `environments/default/version.yaml` 文件，配置 bcs-services-stack charts version 为 `1.29.6`：
``` bash
sed -i 's/bcs-services-stack:.*/bcs-services-stack: "1.29.6"/' environments/default/version.yaml
grep bcs-services-stack environments/default/version.yaml  # 检查修改结果
```
预期输出：
>``` yaml
>  bcs-services-stack: "1.29.6"
>```

更新 bcs-services-stack：
``` bash
helmfile -f 03-bcs.yaml.gotmpl -l name=bcs-services-stack sync
```

等待命令执行完毕，结尾输出如下即为更新成功：
>``` plain
>UPDATED RELEASES:
>NAME                CHART                        VERSION
>bcs-services-stack  blueking/bcs-services-stack  1.29.6
>```
