## 持续集成平台

### bk-ci-3.0.15-beta.12

这是 **重要更新**，主要修复了新建项目 404 等问题，强烈建议更新。
<!-- 版本日志见 GitHub_URL 。-->

登录到 **中控机**，先更新 helm 仓库缓存：
``` bash
helm repo update
```
检查仓库里的版本：
``` bash
helm search repo bk-ci --version 3.0.15-beta.12
```
预期输出如下所示：
>``` plain
>NAME            CHART VERSION   APP VERSION  DESCRIPTION
>blueking/bk-ci  3.0.15-beta.12  略           略
>```

接下来开始升级了。

先进入工作目录：
``` bash
cd $INSTALL_DIR/blueking/  # 进入工作目录
```

修改 `environments/default/version.yaml` 文件，配置 bk-ci charts version 为 `3.0.15-beta.12`：
``` bash
sed -i 's/bk-ci:.*/bk-ci: "3.0.15-beta.12"/' environments/default/version.yaml
grep bk-ci environments/default/version.yaml  # 检查修改结果
```
预期输出：
>``` yaml
>  bk-ci: "3.0.15-beta.12"
>```

更新 bk-ci：
``` bash
helmfile -f 03-bkci.yaml.gotmpl -l name=bk-ci sync
```

等待命令执行完毕，结尾输出如下即为更新成功：
>``` plain
>UPDATED RELEASES:
>NAME   CHART           VERSION
>bk-ci  blueking/bk-ci  3.0.15-beta.12
>```
