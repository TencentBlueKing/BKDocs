# 语法

## 二元操作符语法和使用示例参考下面的表格

| 二元操作符 | 含义     | 示例                                                              |
| ---------- | -------- | ----------------------------------------------------------------- |
| ==         | 等于     | ${v1} == True<br>\${v1} == 1<br>1 == True<br> '\${v1}' == 'hello' |
| !=         | 不等于   | ${v1} != 1<br />0 != True<br> '\${v1}' != 'hello'                 |
| >          | 大于     | ${v1} > 1                                                         |
| <          | 小于     | ${v1} < 1                                                         |
| >=         | 大于等于 | ${v1} >= 1                                                        |
| <=         | 小于等于 | ${v1} <= 1                                                        |
| in         | 包含于   | ${v1} in (1, 2)                                                   |
| notin      | 不包含于 | ${v1} notin (1, 2)                                                |
